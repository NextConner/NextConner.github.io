¶The Feyman Technique ：The Best Way To Learn Anything

  1. 在 Google ‘the best way of learning knowledge’ 的时候看到了这篇文章，介绍了费曼学习方法，还用上了学习任意事物的最佳方法这样的形容词，就点开仔细看一下，确实是很科学的，质朴的学习方法，不过相比现在的快节社会而言，可能大部分人都做不到 (我也是 )

  2. 费曼学习法的四个步骤是：

    1. 确定你要学习/了解的知识点/概念
    2. 假设你将要把这个概念讲述给一个12，3岁刚好具备认知能力的学生听
    3. 鉴别你的理解断层，回到知识源，重新去理解
    4. 回顾并简化这个概念

  3. 停止学习你就会停滞不前，但是我们如何从学习中获得反馈？以及如何学习一个新的主题并且鉴别出自身已有知识的断层所在？

  4. 知识和认知：费曼明白了知道一件事物的名字和了解一件事物的区别，也是他取得成功的重要因素之一。我们多数人都关注了错误的认知方式，就是仅仅是知道一个事物，不会再有多的了解，并且认为这就是全部，学习的过程也就停止在这里；而正确的的认知方式是对一个事物有清晰明白的认识和了解–更加内在的精髓，这才是真正的对事物的认知

声称自己了解了某些东西但是不能够很好地表达出来的人，通常都不了解自己的想法

  1. 费曼学习法步骤：
    1. 将你要学习的知识讲述给12，3岁的小孩子听：
      1. 将你要学习/了解的主题写在白纸上，然后将所有你关于这个主题的了解都写下来就像你将要把它教会给一个小孩子一样。讲述的对象不是一个成人，而是选择具备基础认知水平的小孩子，这样更加能够测试你对于这个主题的理解
      2. 事实表明：我们欺骗自己的一种方式是使用复杂的词汇和术语来表达自己对知识的理解，这一点反而体现了对它缺乏理解–没有形成自己的思维
      3. 当你需要从头到尾写出你关于某个主题的理解并且要让一个小孩子也能理解这个主题，你就得强迫自己加深对它的理解，并且简化观点之间的关联和联系，这样才能表达得更简单
      4. 当你对主题的了解足够深刻的时候，上面的步骤实行起来很容易；反之亦然，只要你的理解出现断层，你就会很痛苦
    2. 回顾主题
      1. 只有当你在理解知识的过程中遭遇了断层 – 忘记了一些重要的概念以至于无法解释它，或者不明白变量之间的相互作用 – 这种情况下你才是真正开始学习
      2. 现在你已经了解了自己的困境所在，返回知识源重新学习直到你能用基础的东西去表达这个主题/概念。只有当你能够不通过复杂的专业术语而是简单的词汇来表达概念的时候，你才可以在别人面前对这个概念进行演示；这是学习过程中所必须的工作，跳过这一步会让你产生学习幻觉
    3. 组织和简化
      1. 到了这一步，你已经累积了一定程度的知识手记/奴笔记，确保你没有错误地从知识源中借用任何专业术语。使用简单的叙事方式阐述这个主题，大声地读出来。如果表述的方式不够简单或者听起来仍然觉得有困惑，充分表明你还需要在知识源的学习上花费更多的精力。
      2. 如果你坚持以这种目标去学习，你会获得一本装满不同主题的活页夹。一两年后你会惊讶于自己了解到了多么丰富的知识。
    4. 传达
      1. 如果你真的想知道自己是否确切地了解了这个主题，将它传达/讲述给一个12，3岁的小孩（之前只是作为场景假设），对于你是否掌握了知识的最终测试就是你将它传达给另一个人的能力
  1. 费曼学习法 不仅是一个学习知识的优良方法，同时也是为你打开不同思维窗口的方式，让你将知识分解并且重组，这个过程毫无疑问会加深你对它的理解
  2. 当你在和别人交谈的过程中发现他们使用了一些你不理解的专业术语来表达，可以考虑让他们试着把你当作12，3岁的小孩，向你解释清楚那些概念
  3. 这样不仅会给你自己的学习过程增压，也会同样作用在他们身上。这种直指问题本身的方式会让你明白：别人可能并不知道自己在讨论什么。
  4. 费曼学习法表明：智力是一个不断成长的过程；Carol Dweck 的研究工作完美地阐释了这一点 The difference between a fixed and growth mindset
