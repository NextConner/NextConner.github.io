Redis 学习笔记(一)

概述：

  • Redis 是一种基于键-值的NoSQL数据库，键是String类型，值可以是由 String,hash,list,set,zset,Bitmaps,HyperLogLog,GEO(地理信息定位)等数据结构和算法组成
  • 因为Redis把数据保存在内存中，因此读写速度很快，而且还可以将数据持久化到本地，即使发生断电或者机器故障也可以恢复数据。
  • 除此之外Redis还提供了键过期，发布订阅，事务，流水线，Lua脚本等附加功能。

Redis 的单线程架构：

  • 在提到Redis 特点的时候，提到Redis的处理速度很快，除了因为数据是内存存储之外，也因为单线程架构编程简单，避免了线程切换和多线程竞争的问题；以及非阻塞IO-IO多路复用实现的网络请求处理，这三点构成了redis 在大部分正确使用的情况下的快速和高效。

    IO多路复用：

    • 在原始的阻塞IO中，进程的操作线性执行，在客户端没有发送数据的情况下，read 操作会一直阻塞直到有数据 write 进来。
